# Artefatos da pesquisa
Tabelas de resultados obtidos por meio da pesquisa de fatores críticos de sucesso e aspectos de qualidade aplicados ao desenvolvimento de software ágil.

## Estrutura de diretórios
```
research-artifacts/
├── README.md                           # Descrição geral da organização do repositório
└── tables/                             # Tabelas utilizadas para a síntese dos resultados da pesquisa
    ├── agile_practices/                # Tabela de síntese das principais práticas ágeis identificadas como impactantes para a 
    |                                       qualidade dos artefatos. Formato em .pdf e em .tex
    ├── metrics/                        # Tabela de síntese das principais métricas de qualidade identificadas. Formato em .pdf 
    |                                       e em .tex
    ├── success_factors/                # Tabela de síntese dos principais fatores críticos de sucesso identificados. Formato em
    |                                       .pdf e em .tex          
    ├── ACM-Reference-Format.bst        # Arquivo para formatação das referências do documento        
    ├── acmart.cls                      # Arquivo para formatação do documento    
    └── references.bib                  # Referências utilizadas na sumarização dos dados nas tabelas 
```
